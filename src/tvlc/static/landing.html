Self-Host veedeeoh on Your Own Hardware

Deploy locally on Docker, Unraid, Synology, or any home server. Zero telemetry, no external database required, and instant local stream proxying.

1

Docker Compose Deployment

Clone the repository and launch your containerized media server in seconds.

git clone https://github.com/ItsCodejac/veedeeoh.git cd veedeeoh docker compose up -d
2

Zero Database Overhead

Uses lightweight JSON persistence for user state and favorites. No PostgreSQL or Redis needed.

# Persistence Volume Mapping volumes: - veedeeoh_data:/root/.local/share/tvlc
